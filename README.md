# Max’s Deli

A responsive website for Max’s Deli in Wayne, NY featuring daily specials, full catering services, and more.

## Features
- Sticky Bootstrap navbar
- Responsive menu
- Daily specials section
- Auto-scrolling image carousel using Swiper.js
